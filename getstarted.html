Glutch Installation

The offical way to install Glutch into your project!



#1 Copy The Stylesheet

Copy the code below and paste it into your header.


<link rel="stylesheet" href="https://glutch.glitch.me/style.css">

Example:


<head> <link rel="stylesheet" href="https://glutch.glitch.me/style.css"> </head>
